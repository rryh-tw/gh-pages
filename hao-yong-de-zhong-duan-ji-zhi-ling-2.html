好用的終端機指令

Rémi

1. more

顯示太多，使用more指令會讓畫面滿一頁時暫停，按空白鍵繼續顯示下一畫面。
ls -al | more
cat input.txt | more 

2. grep

找到所有cpp檔裡，含有cin的內容。
grep cin *.cpp
H.cpp:    // int _T; cin >> _T; while(_T--)
H2.cpp:    cin >> n;
H2.cpp:    for(auto &x:v) cin >> x;
H_1.cpp:#define IOS cin.tie(nullptr)->sync_with_stdio(false);
ls | grep A
ls的結果，丟給grep塞選。
 

3. &

可以讓run A指令在背景跑，繼續讓他跑輸入fg(foreground)
royyaha%run A &
[1] 45598
royyaha% jobs
[1]+ Running run A &
royyaha% fg
[1]  + continued  run A

4. echo $PATH

可以查看環境設定的目錄有哪些
